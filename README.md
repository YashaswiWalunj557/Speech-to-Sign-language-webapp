## 🚀 About The Project

**Real-Time Speech to Sign Language Translation** is an intelligent system designed to bridge the communication gap between hearing and speech-impaired individuals and the general public.

The application captures spoken input, processes it using speech recognition, and translates it into corresponding **sign language gestures or animations** in real time. This makes communication more inclusive, interactive, and efficient.

---

## ✨ Features

* 🎤 Real-Time Speech Recognition
* 🧏‍♂️ Speech to Sign Language Conversion
* ⚡ Instant Output with Minimal Delay
* 🌐 Web-Based Interface (Accessible Anywhere)
* 🧠 AI-Based Processing for Accurate Translation
* 📚 Predefined Dataset of Sign Gestures
* 🔄 Continuous Listening Mode
* ➕ Easy to expand with new signs and commands

## ⚙️ Installation

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/YashaswiWalunj557/Speech-to-Sign-language-webapp.git

# Navigate to project folder
cd Speech-to-Sign-language-webapp

# Install required libraries
pip install -r requirements.txt

# Run the application
python app.py
```
